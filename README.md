# Takeaway Skill

English | [简体中文](./README.zh-CN.md)

An open-source gallery shell for showcasing web experiments with a branded hero banner and an editable entry framework.

This repository keeps:

- the branded banner
- the list layout framework
- the local view style
- one empty template row

This repository does not include:

- real case studies
- private research archives
- source material packages
- private links
- local absolute paths

## Structure

- `site/index.html` - main entry page
- `site/assets/` - public-facing visual assets
- `site/ui/` - local UI styles

## Language Strategy

- Branding copy can stay in Chinese.
- Structural UI should stay in English.
- Documentation uses English-first with a Chinese companion file.

## Quick Start

1. Open `site/index.html`.
2. Replace the banner copy and image if needed.
3. Duplicate the template row and add your own projects.
4. Replace the placeholder links with real destinations.
